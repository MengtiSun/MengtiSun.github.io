Basic SVG Tags & Attributes

One of the most important tag in SVG is path.

These commands are availble in the attribute d.

path commands
M = moveto(M X,Y)
L = lineto(L X,Y)
H = horizontal lineto(H X)
V = vertical lineto(V Y)
C = curveto(C X1,Y1,X2,Y2,ENDX,ENDY)
S = smooth curveto(S X2,Y2,ENDX,ENDY)
Q = quadratic Belzier curve(Q X,Y,ENDX,ENDY)
T = smooth quadratic Belzier curveto(T ENDX,ENDY)
A = elliptical Arc(A RX,RY,XROTATION,FLAG1,FLAG2,X,Y)
Z = closepath()

M, L, H, Z and their parameters are easy to understand. FLAG1 and FLAG2 inside of A need to be explained.

FLAG1 has only two values: 0 and 1. 1 means the larger curve plotted will be chosen while 0 means the smaller one/

FLAG2 also has only two values: 0 and 1. If it is 1, the curve will be plotted in 'clockwise' direction. On contrary, the curve will in 'counter clockwise' direction.

Another important attribute is stroke, including stroke, stroke-width, stroke-linecap, stroke-linejoin, stroke-miterlimit, stroke-dasharray, stroke-dashoffset, stroke-opacity.

stroke-dasharray='num1,num2,num3,num4...' can create dashed line. The 2nth number is the solid line length while the (2n+1)th number represents the length of empty space.

stroke-dashoffset='num' is the offset of the start of the stroke.

Letting Your Graphics Move

Applying stroke-dasharray and stroke-dashoffset to let the stroke move.

The SVG code of my Logo (HTML part)
#s, #u, #n, #rabbit_face, #rabbit_mouse {
stroke: #9966ff;
stroke-width: 40;
fill: #EFEFEF;
}
#rabbit_eyes {
stroke: #9966ff;
stroke-width: 40;
fill: #9966ff;
}
#s, #u, #n {
stroke-dasharray: 5300;
-webkit-animation: s-stroke 3s ease-in-out;
animation: s-stroke 3s ease-in-out;
}
#rabbit_face {
stroke-dasharray: 2000;
-webkit-animation: rabbit-face-stroke 1s ease-in-out 3s;
animation: rabbit-face-stroke 1s ease-in-out 3s;
-webkit-animation-fill-mode: both;
animation-fill-mode: both;
}
